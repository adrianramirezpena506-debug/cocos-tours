# Coco's Tours — Premium Local Redesign

Rediseño premium, bilingüe y responsive del sitio de Coco's Tours, construido con Vite, React, TypeScript y Tailwind CSS.

## Desarrollo local

```bash
npm install
npm run dev
```

## Validación

```bash
npm test
npm run lint
npm run build
```

El proyecto utiliza consultas estructuradas por WhatsApp. El inicio de sesión, registro, pagos, wishlist y seguimiento de reservas permanecen como funciones externas del sitio publicado de Coco's Tours.

Consulta [`docs/content-audit.md`](docs/content-audit.md) para conocer las fuentes, decisiones editoriales y datos verificados.
